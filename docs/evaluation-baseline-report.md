# Evaluation Baseline Report Template

Generated by `python scripts/run_eval.py --report`.

## Fields

| Field | Description |
|-------|-------------|
| `report.total` | Scenario count |
| `report.passed` | Passing scenarios |
| `report.pass_rate` | passed / total |
| `report.metrics.categories_covered` | Distinct eval categories |
| `comparison.status` | `ok`, `regression`, `ratcheted`, `created` |

## Ratchet Policy

Update `eval/baseline.json` only with:

```bash
python scripts/run_eval.py --ratchet --reason "Explain why metrics improved"
```

Unexplained regressions block CI when `comparison.status == regression`.

## Baseline Location

Committed baseline: [eval/baseline.json](../eval/baseline.json)

## CI Integration

GitHub Actions workflow `.github/workflows/ci.yml` runs pytest and baseline comparison on every push/PR.
